## рҹҡҖ Getting Started

### Prerequisites

- **Node.js** v18 or higher
- **npm** v9 or higher

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/aayush2411ghoghari-sketch/LIFEBRIDGE-AI.git
cd LIFEBRIDGE-AI

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

The app will be available at **http://localhost:5173**

### Build for Production

```bash
# Create optimized production build
npm run build

# Preview the production build locally
npm run preview
```

## рҹ”„ CI/CD Pipeline

LifeBridge AI uses **GitHub Actions** for continuous integration and deployment.

```mermaid
flowchart LR
    A[рҹ‘ЁвҖҚрҹ’» Developer Push] --> B[рҹ“Ұ Install & Lint]
    B --> C{вң… TypeScript Check}
    C -->|Pass| D[рҹҸ—пёҸ Build with Vite]
    C -->|Fail| E[вқҢ Block Merge]
    D --> F{Branch?}
    F -->|main| G[рҹҡҖ Deploy to GitHub Pages]
    F -->|other| H[рҹ“Ө Upload Build Artifact]
    
    style A fill:#1a1a40,color:#e0e0e0
    style G fill:#0d4f3c,color:#e0e0e0
    style E fill:#4a0d0d,color:#e0e0e0
```

### Pipeline Jobs

| Job | Trigger | Description |
|---|---|---|
| рҹ“Ұ **Install & Lint** | All pushes & PRs | Installs deps, runs TypeScript type check |
| рҹҸ—пёҸ **Build** | After lint passes | Creates production Vite bundle |
| рҹҡҖ **Deploy** | Push to `main` only | Auto-deploys to GitHub Pages |
